# Algoritmi e Strutture Dati — Pratica d'Esame

Questo repository contiene le mie soluzioni personali ad alcuni appelli passati
del corso di **Algoritmi e Strutture Dati**.

Le soluzioni sono scritte principalmente in stile esame, quindi con attenzione a:

- idea dell'algoritmo;
- pseudocodice;
- correttezza;
- complessità temporale e spaziale;
- eventuali osservazioni utili per evitare errori comuni.

Il repository è pensato come raccolta ordinata di esercizi svolti durante la
preparazione dell'esame.

## Fonte degli appelli

Gli appelli sono presi dalla pagina ufficiale del corso:

```text
https://www.math.unipd.it/~baldan/Algoritmi/Exams/
```

I PDF originali non sono inclusi nel repository.

## Nota

Questo repository non è materiale ufficiale del corso.

Le soluzioni sono personali, non approvate dai docenti, e potrebbero contenere
errori o imprecisioni. Lo scopo è raccogliere il mio lavoro di studio e rendere
più chiaro il ragionamento dietro ogni esercizio.

## Struttura del repository

Ogni cartella corrisponde a un appello d'esame.

```text
YYYY-MM-DD/
├── README.md
├── domanda-a.md
├── domanda-b.md
├── esercizio-1.md
└── esercizio-2.md
```

Ogni cartella contiene:

- `README.md`: panoramica dell'appello;
- `domanda-a.md`: soluzione della Domanda A;
- `domanda-b.md`: soluzione della Domanda B;
- `esercizio-1.md`: soluzione completa dell'Esercizio 1;
- `esercizio-2.md`: soluzione completa dell'Esercizio 2.

## Formato delle soluzioni

Per gli esercizi più lunghi uso di solito questa struttura:

```text
1. Traccia
2. Idea
3. Pseudocodice
4. Correttezza
5. Complessità
6. Osservazioni
```

Per le domande teoriche uso una forma più compatta, mantenendo comunque
definizioni, spiegazioni e dimostrazioni quando richieste.

## Argomenti ricorrenti

Gli appelli raccolti includono esercizi su:

- notazione asintotica e ricorrenze;
- divide et impera;
- heap;
- tabelle hash;
- alberi binari di ricerca;
- alberi arricchiti;
- greedy;
- Huffman;
- programmazione dinamica e memoizzazione;
- correttezza e analisi della complessità.
